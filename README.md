Contact Manager
Description
Contact Manager est une application web de gestion de contacts développée avec Laravel. Elle permet d'organiser vos contacts par groupes, de rechercher, filtrer et gérer efficacement vos relations professionnelles et personnelles.

 Fonctionnalités
 Gestion des Groupes
 Créer un groupe - Formulaire de création avec validation

 Modifier un groupe - Mise à jour du nom du groupe

 Supprimer un groupe - Suppression propre avec détachement des contacts

 Voir la liste des groupes - Interface moderne avec sidebar

 Gestion des Contacts
 Ajouter un contact - Formulaire avec nom, email, téléphone et sélection de groupe

 Modifier un contact - Édition des informations complètes

 Supprimer un contact - Confirmation avant suppression

 Voir la liste des contacts - Tableau responsive avec pagination

 Rechercher un contact - Recherche par nom (insensible à la casse)

 Filtrer par groupe - Affichage sélectif des contacts

 Interface Utilisateur
 Design moderne - Couleurs noir, blanc, rouge

 Sidebar navigation - Navigation intuitive et responsive

 Messages flash - Retour visuel sur les actions

 Responsive design - Compatible mobile et tablette
